# LittleLights — Art Direction Guide

This document defines the visual identity of LittleLights.
All artwork must follow these rules to maintain consistency across AI generation, human artists, and future agents.

---

## Core Visual Philosophy

LittleLights should feel:

- Warm
- Safe
- Gentle
- Encouraging
- Peaceful
- Hopeful

This is NOT an action game.
This is a calm learning experience.

Primary references:

- Children's Bible illustrations
- Montessori children's books
- Calm storybook illustration styles
- Modern flat children's apps

Avoid:

- High contrast aggressive styles
- Anime style
- Comic superhero style
- Hyper realism
- Dark or moody tones

---

## Target Audience

Age: 4–7 years old

Design implications — everything must be:

- Easy to read
- Easy to recognize
- Emotionally safe
- Not visually overwhelming
- Clear shapes
- Clear silhouettes

> **Rule:** If a child cannot identify the object in 1 second, the design is wrong.

---

## Illustration Style

**Style type:** Soft Flat Storybook Style

Hybrid between flat illustration + soft watercolor softness.

Rules:

- Soft edges
- Rounded shapes
- Minimal sharp corners
- Minimal hard outlines
- Gentle shadowing only
- Subtle gradients allowed

Avoid:

- Realistic textures
- Noise textures
- Complex shading
- Photorealism

Line style: optional very soft outline, 2–4 px equivalent. Color outline allowed instead of black.

## Color System

### Primary palette

| Name | Hex |
|---|---|
| Warm cream (background) | `#FFF8EC` |
| Sky blue | `#C8E6F5` |
| Soft green | `#B5D99C` |
| Accent orange | `#F4A261` |
| Soft brown | `#C9A27C` |

### Secondary neutrals

| Name | Hex |
|---|---|
| Warm grey | `#DAD7CD` |
| Soft shadow | `#EAE7E1` |

### Rules

- **Never use pure black.** Use dark brown `#5B4636` instead.
- **Never use pure white.** Use warm white `#FFF8EC` instead.
- All colors across stories must use the same saturation level and warmth.

---

## Character Design Rules

Characters must feel: friendly, kind, safe, approachable.

Rules:

- Large heads
- Simple eyes (small oval or dot: `● ●` or `◉ ◉`)
- Minimal facial detail
- Soft smiles (simple curved line — no detailed lips)
- No sharp teeth
- No intense expressions

Body proportions: 2–3 heads tall (storybook style).

---

## Noah Character Rules

Noah should be a friendly grandfather figure.

Design:

- Short beard
- Warm smile
- Simple robe
- Sandal colors
- Soft brown hair
- Light tan skin

Avoid: realistic aging, wrinkle detail, dramatic biblical depictions.

Mood: welcoming and joyful.

---

## David Character Rules

David should feel like a brave but gentle young boy.

Design:

- Small frame (young shepherd)
- Simple tunic
- Soft confident smile
- Sling in hand
- Warm olive skin

Goliath should feel: large but cartoonish, not scary.

Design:

- Very large rounded body
- Simple armor shapes
- Slightly confused expression (not angry/threatening)

Mood: David = hopeful courage. Goliath = cartoonish size, not danger.

---

## Animal Design Rules

Animals must be: cute, rounded, simplified.

Rules:

- Large eyes
- Rounded snouts
- Simplified shapes
- No realism

Examples:
- **Lion:** Rounded mane like a soft cloud.
- **Elephant:** Large soft ears, small eyes, short trunk curve.
- **Giraffe:** Rounded spots, gentle smile.

Pairs must look clearly identical — either mirrored or a slight pose change.

---

## Environment Style

Backgrounds must be: calm, soft, non-distracting.

Rules:

- Simple gradients
- Large soft shapes
- Minimal detail
- No texture noise
- Sky: soft gradient
- Clouds: round soft shapes
- Ground: rolling hills

Avoid complex landscapes.

---

## UI Style

Buttons: rounded rectangles, minimum radius 20px equivalent.

Colors:
- Primary action: orange `#F4A261`
- Secondary: soft blue `#C8E6F5`

Icon style: simple, single color, no detail.

---

## Lighting Rules

Lighting must be: soft, diffuse, warm.

Avoid: hard shadows, dramatic lighting.

---

## Animation Feel

Motion must feel: gentle, playful, not fast.

| Element | Duration |
|---|---|
| Small UI transitions | 200–300ms |
| Reward reveals | 400–700ms |
| Celebration sequences | 800–1200ms |

Avoid fast snappy animation.

| SFX clips | MP3 | 0.5–2 sec, 44.1 kHz |

---

## Cross-Story Consistency Rules

- All animals across all stories must use the same eye style, shading style, and color saturation.
- All Noah images must reuse the same character design — never redesign mid-project.
- Cutscenes must reuse the same ark design seen in gameplay.
- When a new story (David, Jonah, Adam) is added, match the existing palette exactly — do not introduce new hues.

---

## Forbidden Styles

Never generate:

- Anime
- Realistic painting
- 3D renders
- Pixel art
- Dark fantasy
- Comic book style
- Horror tone

---

## AI Prompt Consistency Rule

Every AI-generated asset prompt must include this base phrase:

> **"soft children's storybook illustration, warm pastel colors, rounded shapes, gentle lighting, simple friendly character design"**

Add subject-specific detail after this base phrase.

---

## The Golden Rule

> If an image looks **exciting** → it is **wrong.**
> If an image looks **calming** → it is **correct.**

LittleLights is about: **peace, faith, gentleness, hope.**
